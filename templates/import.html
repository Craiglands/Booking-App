Import Bookings from Excel

Upload an Excel file (.xlsx or .xls) with the following columns (at minimum):

  • date, service, time, name (required)
  • Optional: tel, guest_email, voucher, notes, dietary, paid, guests, room, surname, filling, bread, collection_time
  • For packed lunch guest meals:
    • If only one guest, use columns filling, bread, dietary (will be used for guest 1).
    • For multiple guests, use numbered columns: guest_1_filling, guest_1_bread, guest_1_dietary, guest_2_filling, etc.
  • Note: For packed lunch, the collection_time column will be used as the time. The time column should contain the collection time or be left blank – it will be ignored.
